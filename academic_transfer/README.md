# Автоматизация процессов перевода и восстановления студентов ЯГТУ

## Описание проекта

Веб-приложение для автоматизации процессов перевода и восстановления студентов. Приложение позволяет загружать учебные планы и ведомости с оценками, автоматически сопоставлять дисциплины, выявлять задолженности и формировать рекомендации по перезачету/переаттестации.

## Функционал

- Загрузка учебных планов и ведомостей (XLSX, CSV)
- Автоматическое сопоставление дисциплин по названиям
- Ручное сопоставление несопоставленных дисциплин
- Формирование итогового списка по категориям:
  - Перезачитываются
  - На переаттестацию
  - Требуют изучения
- Экспорт результатов в CSV

## Технологии

- Python 3.8+
- Flask
- pandas
- openpyxl
- HTML/CSS/JavaScript
